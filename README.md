# BioScanner Pro
**A High‑Throughput 128‑Well Plate Image Analysis Tool for Biological Assays**

第十一届全国大学生生命科学竞赛（科学探究类）源码

---

## 简介
BioScanner Pro 是一款面向生命科学实验的高通量图像分析软件，主要用于**128 孔板/微流控芯片**的快速定量分析，适用于微生物筛选、酶活检测、比色法实验等场景。

核心特点：
- ✅ 一键加载任意分辨率孔板图像
- ✅ 多通道自动标曲（R/G/B/H/SV/Gray），自动选最优通道（R² 最高）
- ✅ 三点自动生成 16×8 网格，精准定位 128 孔
- ✅ 全孔颜色定量 → 浓度换算 → 结果表格化展示
- ✅ 一键导出 CSV，便于统计与二次分析
- ✅ 跨平台：Windows / macOS / Linux 均可运行
- ✅ GitHub Actions 自动打包，开箱即用

---

## 功能一览
- **图像预处理**：高斯模糊去噪、HSV/灰度转换
- **标准曲线校准**：用户标记标准点 → 线性回归 → 自动选最优通道
- **128 孔板网格映射**：三点（A1/A16/H1）自动生成整板网格
- **高通量分析**：逐孔提取颜色 → 浓度计算 → 实时显示结果
- **结果导出**：表格预览 + CSV 导出
- **跨平台打包**：GitHub Actions 自动构建发布可执行文件

## 使用流程（三步搞定）
1. **Calibration（校准）**：
   - 加载含标准浓度点的图像
   - 点击标记已知浓度位置并输入浓度
   - 自动拟合曲线，选择最优通道

2. **Scan（扫描）**：
   - 加载 128 孔板图像
   - 依次标记 A1、A16、H1 三点
   - 自动生成网格并逐孔分析

3. **Export（导出）**：
   - 查看实时结果表格
   - 一键导出 CSV，用于后续绘图/统计
